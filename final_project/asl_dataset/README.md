# ASL Training Dataset

This folder is reserved for the American Sign Language (ASL) alphabet dataset used
to train all models in this project.

## Dataset Description

The training data consists of static images representing ASL alphabet hand gestures.
Each class corresponds to one alphabet letter and is organized into class-wise
subfolders.

The dataset contains only static gestures and does not include dynamic signs
that require motion (such as J and Z).

## Source

The ASL alphabet dataset used in this project is publicly available and can be
downloaded from Kaggle:

https://www.kaggle.com/datasets/datamunge/sign-language-mnist

The dataset is not included in this repository to reduce repository size and to
comply with dataset licensing guidelines.
